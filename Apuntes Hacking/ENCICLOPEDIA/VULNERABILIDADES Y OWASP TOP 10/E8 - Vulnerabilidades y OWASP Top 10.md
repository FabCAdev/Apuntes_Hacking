Índice central para el análisis y explotación de fallas en aplicaciones web bajo el estándar OWASP.

## 🗂️ Nodos de Conocimiento (Conceptos)
- [[Vulnerabilidad - SQL Injection (SQLi)]]
	- *Descripción:* Inyecciones basadas en errores, UNION, booleanas y basadas en tiempo manual y con sqlmap.
- [[Vulnerabilidad - Cross-Site Scripting (XSS)]]
	- *Descripción:* Ejecución de código del lado del cliente (Reflejado, Almacenado y basado en DOM).
- [[Vulnerabilidad - Local File Inclusion (LFI)]]
	- *Descripción:* Lectura de archivos internos del sistema mediante abusos de rutas relativas.
- [[Vulnerabilidad - Remote File Inclusion (RFI)]]
	- *Descripción:* Inclusión remota de código en servidores vulnerables desconfigurados.
- [[Técnica - Log Poisoning para RCE]]
	- *Descripción:* Envenenamiento de archivos de registro (Apache/SSH) para forzar ejecución de comandos.
- [[Vulnerabilidad - XML External Entity (XXE)]]
	- *Descripción:* Abuso de analizadores de entidades externas XML para lectura de datos o SSRF.
